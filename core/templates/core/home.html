  • Make a POST request to
    api/emails/
    To add to the email list
    • The required parameter here is email
    • A successful request here returns something like
      • {"email": "newer070504249272@eefsrgd.khjbv"}
  • Make a DELETE request to
    api/emails/{number}
    Where number is the id of the email to be deleted.
    A successful request here returns nothing, you can always GET the list to be sure only what is needed is present
